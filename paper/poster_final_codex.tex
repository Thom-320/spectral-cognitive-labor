% Question
\noindent
\eyebrow{Pregunta central}
{\sffamily\fontsize{38}{47}\selectfont
¿Cuál es la referencia espectral correcta para comparar particiones humanas cuando el tablero cuadrado induce un eigenespacio de Fiedler degenerado, y puede una señal geométrica temprana anticipar la especialización axial estable?\par}
\vspace{0.35cm}
\softtext{Dos jugadores sin comunicación exploran una grilla $8\times8$. El análisis se concentra en rondas ausentes para particiones y usa la tabla completa para transferencia descriptiva a desempeño.}
